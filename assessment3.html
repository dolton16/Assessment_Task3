Web Attacks Overview

Attack Overview
Man in the Middle (MITM)

Image source: Imperva 		A man in the middle (MITM) attack is a general term for when a perpetrator positions himself in a conversation between a user and an application—either to eavesdrop or to impersonate one of the parties making it appear as if a normal exchange of information is underway.
Reference/More Information: Imperva
Drive-By Download Attacks

Image source: Norton 		The reach of the internet isn’t slowing down and neither is the danger of drive-by downloads. With an average number of 640000 new internet users each day, hackers can expose more and more people to potentially dangerous websites.
Reference/More Information: Norton
Phishing and Spear-Phishing Attacks

Image source: Hoxhunt 		Phishing and spear phishing are very common forms of email attack designed to you into performing a specific action — typically clicking on a malicious link or attachment. The difference between them is primarily a matter of targeting.
Reference/More Information: Barracuda
Password-Based Attacks

Image source: Medium 		A password attack is an attempt by hackers to gain unauthorized access to accounts or systems by utilizing various methods to determine the right password such as guessing, brute force, or exploiting vulnerabilities. The purpose is to compromise security and obtain sensitive information.
Reference/More Information: Medium
SQL Injection

Image source: Simplilearn 		SQL Injection is a code-based vulnerability that allows an attacker to read and access sensitive data from the database.
Reference/More Information: Simplilearn
Clickjacking

Image source: Ping Identity 		The most common type of clickjacking attacks are called overlay attacks. These are made possible by invisible frames (iframes) in which attackers cover a legitimate-looking web page with an interface that the user cannot see for malicious purposes.
Reference/More Information: Ping Identity
Cross-Site Scripting (XSS)

Image source: Imperva 		Cross-Site Scripting (XSS) attacks are a type of injection in which malicious scripts are injected into otherwise benign and trusted websites.
Reference/More Information: OWASP
Cross-Site Request Forgery (CSRF)

Image source: PortSwigger 		Cross-site request forgery (CSRF) is an attack that tricks a user's browser into sending a malicious HTTP request to another website. This malicious HTTP request looks like it was sent by the user but it actually comes from the attacker.
Reference/More Information: Atatus
Session Hijacking

Image source: Invicti 		Session hijacking is an attack where a user session is taken over by an attacker.
Reference/More Information: Invicti
Session Fixation Attack

Image source: GeeksforGeeks 		A session fixation attack is a type of remote code execution attack which is used to exploit software designed with the web-server Session Management feature.
Reference/More Information: GeeksforGeeks
